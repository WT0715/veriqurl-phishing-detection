# VeriqURL

VeriqURL is a lightweight URL-based phishing detection and awareness prototype.

## Features

- Random Forest phishing URL prediction
- Character-level TCN phishing URL prediction
- Combined model comparison
- Risk level output
- User-friendly explanation
- Recommendation and awareness tip
- Lightweight metrics comparison

streamlit run app.py
